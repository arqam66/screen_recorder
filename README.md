# Arqam Recorder

A free browser-based screen recorder with face cam overlay, voice recording, and instant local download. No server, no upload, no account required — everything runs entirely in the browser.

---

## Features

- **Screen only** — Record any tab or window with microphone
- **Camera only** — Record face cam with voice
- **Screen + Camera** — Face overlay on screen recording
- **Live timer & audio meter** — Real-time recording feedback
- **Quality presets** — 1–8 Mbps bitrate control
- **Camera controls** — Adjustable position, shape, and size
- **Annotation tools** — Laser pointer and pen drawing
- **Screenshot capture** — PNG snapshot during recording
- **Preview before download** — Review then save or discard
- **Keyboard shortcuts** — `Space` pause, `Esc` stop, `S` screenshot

---

## Usage

| Mode | Steps |
|------|-------|
| **Screen** | Select mode → Start → pick a Chrome Tab → allow mic → record |
| **Camera** | Select mode → Start → allow camera & mic → record |
| **Screen + Camera** | Select mode → Start → pick a tab → allow camera & mic → record |

Click **Stop** to preview and download the recording as `.webm`.
